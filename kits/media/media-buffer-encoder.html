Constructor and Destructor¶

BMediaBufferEncoder()

BMediaBufferEncoder::BMediaBufferEncoder()
BMediaBufferEncoder::BMediaBufferEncoder(const media_format *outputFormat)
BMediaBufferEncoder::BMediaBufferEncoder(const media_codec_info *mci)

The constructor sets up the BMediaBufferEncoder. If you use the empty form of the constructor, you’ll have to call SetTo() to establish the format to be encoded before calling Encode(). The second form accepts a format() structure, inFormat, that indicates the type of media data that will be input into the encoder. The third form of the constructor accepts a codec() structure, mci, that determines which codec should be used.

Note

If you use either the format() or codec() form of the constructor, you must call InitCheck() to ensure that construction was successful before using any other functions in this class.

Member Functions¶

EncodeToBuffer()

status_t BMediaBufferEncoder::EncodeToBuffer(void *outputBuffer, size_t *outputSize, const void *inputBuffer, int64 frameCount, media_encode_info *info)

Encodes a chunk of media data from the input buffer by inputBuffer, which contains frameCount frames of data. The encoded data is written into the buffer indicated by outputBuffer. On return, outputSize indicates how many bytes of data were written into the buffer. The encode() structure info is used on input to specify encoding parameters.

Return Code

Description

B_OK

No error.

Other errors.

The encoder’s Encode() function can return errors.

InitCheck()

status_t BMediaBufferEncoder::InitCheck() const

Returns a status_t value indicating whether or not construction was successful. You must call this function after construction before calling any other BMediaBufferEncoder functions.

Return Code

Description

B_OK

The constructor was successful.

Other errors.

See BMediaEncoder’s SetTo() function.
